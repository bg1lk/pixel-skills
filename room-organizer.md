# 房间整理分类助手

打开 PixelOS 桌面：

```bash
start "" "$HOME/.claude/skills/pixel-os/app.html"
```

告诉用户像素系统已打开，双击桌面上的「整理」图标即可使用房间整理助手。

功能说明：
- 选择区域（衣柜、书桌、厨房、客厅等）获取专属整理指南
- 物品分类系统：衣服按季节/类型/颜色分类，桌面按功能区规划
- 摄像头拍照记录整理前后对比
- 整理进度追踪，完成任务获得成就
- 内置收纳技巧和断舍离建议
