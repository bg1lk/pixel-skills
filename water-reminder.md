# 智能喝水追踪器

打开 PixelOS 桌面：

```bash
start "" "$HOME/.claude/skills/pixel-os/app.html"
```

告诉用户像素系统已打开，双击桌面上的「喝水」图标即可使用智能喝水追踪器。

功能说明：
- 开启摄像头后，点击画面中的水杯/水瓶注册颜色
- 系统自动追踪水杯位置，检测到喝水动作时自动记录
- 也可手动添加饮水量
- 30分钟未喝水会发出提醒
